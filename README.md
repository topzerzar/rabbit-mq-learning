#rabbit-mq-learning

install dependencies
```
npm install
```

start docker
```
docker-compose up
```

shell to docker container
```
docker exec -it [docker ContainerName or ContainerID] sh
```

goto path app
```
cd usr/app
```

start receiver
```
node receiver.js
```

start sender
```
node sender.js
```
